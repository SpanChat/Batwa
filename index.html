Setup
Connect your Google Sheet once. Data lives there forever — works on any device, any browser.

Step 1 — Create Sheet

Open sheets.new in your browser → name it Finance OS.

Step 2 — Apps Script

In the Sheet: Extensions → Apps Script → delete all → paste Claude's script → change the token → Save → Deploy → New deployment → Web app → Me → Anyone → Deploy → copy the URL.

Step 3 — Connect

Paste your Web app URL and your secret token. These are saved locally — you won't need to enter them again.
